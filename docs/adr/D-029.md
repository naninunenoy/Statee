# D-029 シナリオ内の例外はすべて StandardError で raise する

- **決定**: ScenarioRunner が Ruby へ投げる例外(コマンドのエラー応答・assert 失敗・
  引数形式エラー)は、種類を問わずすべて `StandardError` で raise する。
- **背景**: ChibiRuby 1.5.0 の公開 API で取得できる例外クラスは `ExceptionClass` と
  `StandardErrorClass` のみ(`ArgumentError` / `RuntimeError` を名前で引く API は internal)。
  シナリオ側の異常系ハンドリングは `rescue => e`(StandardError 捕捉)で足りる。
- **トレードオフ**:
  - (+) 公開 API のみに依存し、ライブラリ更新に強い
  - (−) シナリオから例外の種類で分岐できない(必要になったら独自例外クラスを定義する)
- **状態**: ✅ 実装済み(ScenarioRunner)。全99テスト緑
