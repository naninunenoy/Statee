# D-068 サンプルゲームのディレクトリを game/ から samples/ にリネーム

## 決定

サンプルゲーム群を置くトップレベルディレクトリを `game/` から **`samples/`** にリネームする。
各ゲームの Statee State パス(`game/board` 等の名前空間)はディレクトリと無関係な
公開 API なので変更しない。

## 背景

- `game/` は D-018 でスイカゲーム用に温存した名前だが、現在は Reversi / RogueGame /
  ShootingGame / RaidBoss / TodoApp と複数のサンプルが同居しており、単数形が実態と合わない
- ここに置かれるものは「フレームワークの動くデモ」(docs/USING.md 冒頭で明言)であり、
  `samples/` の方が役割(参照実装であって本体ではない)を正しく伝える

## トレードオフ

- 過去の ADR(docs/adr/)内の `game/` 参照は歴史的記録としてそのまま残る。
  現在のパスと食い違うが、記録の改変はしない
- `.claude/settings.local.json` に蓄積された旧パスの許可エントリは無効化される(実害なし)

## 状態

採用。`git mv` + 参照置換(docs / skills / tools / tests の csproj / slnx)済み。
`samples/` 配下の全 slnx でビルド・テスト緑を確認済み。
