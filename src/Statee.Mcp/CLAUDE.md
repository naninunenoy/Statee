# Statee.Mcp 開発指針

- **極力変更しない**(D-001)。ゲーム対応・コマンド追加は CLI 側で吸収する。
  ここを変えたくなったら、まず CLI で実現できないか疑う
- **セッション中は再ビルドできない**: Claude Code セッションがこの exe を
  ロックしている。ソリューション全体ビルドではなくプロジェクト単位でビルドすることで
  巻き込まれを回避する。変更した場合はセッション再起動で反映
- ツールは汎用 1 ツール(`statee_cli`)を維持する。コマンドごとのツール分割はしない
- stdout は MCP の JSON-RPC 専用。ログ・診断出力は stderr のみに出す
