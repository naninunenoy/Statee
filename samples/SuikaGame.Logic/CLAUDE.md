# SuikaGame.Logic 開発指針

- **外部 tick 駆動・壁時計禁止**(D-023)。`DateTime.Now` / `Task.Delay` /
  自前スレッド・タイマーを持ち込まない。時間は `Tick(delta)` でだけ進む
- **物理・位置を持ち込まない**(D-011, D-024)。フルーツの位置は Godot 物理が所有する。
  ここに入れてよいのは「報告を受けて判定する」規則だけ
- **Godot に依存させない**。Godot 型が必要になったらそれは Godot 層の仕事
- 決定論を守る: 乱数はコンストラクタのシード由来の1系統のみ。
  静的な `Random.Shared` 等を使わない(AI 動作確認の再現性の要)
- ゲームオーバー後は盤面凍結(入力を無視)。この不変条件を崩す機能を足すときは
  D-024 を更新する
- ルール変更(スコア表・種類数・猶予時間)は D-024【暫定】の更新とテストの同時修正で行う
